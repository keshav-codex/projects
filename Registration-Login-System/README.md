# Student Registration and Login System

A frontend learning project created using HTML, CSS, Bootstrap and JavaScript.

This project is created to practice frontend development concepts like form validation, Local Storage, JSON data handling and JavaScript logic.

## Features

- Student registration
- Automatic registration number generation
- Login system
- Forgot password option
- Profile photo upload
- Password show/hide option
- Client-side validation
- Dashboard profile display

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- JSON
- Local Storage

## Project Type

Frontend only application.
No backend server or database is used.
