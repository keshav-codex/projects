# Technical Information

## JavaScript Files
The project uses JavaScript for validation and application logic.

## registration.js

Purpose:
Handles student registration process.

Main work:
- Validate form fields
- Check email and mobile format
- Check password rules
- Validate profile photo
- Generate registration number
- Store student data in Local Storage

## login.js

Purpose:
Handles student login.

Main work:
- Check registration number
- Verify password
- Save current user
- Redirect to dashboard

## dashboard.js

Purpose:
Handles dashboard functions.

Main work:
- Display student information
- Display profile photo
- Check login status
- Logout user

## forget-password.js

Purpose:
Handles password reset process.

Main work:
- Verify user details
- Update password
- Save updated data

## CSS File

## style.css

A common CSS file is used for all pages.
It handles:
- Page layout
- Card design
- Form styling
- Buttons
- Profile image
- Responsive design
