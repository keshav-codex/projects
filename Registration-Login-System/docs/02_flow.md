# Application Flow

## User Flow

The application works in the following steps:

## 1. Home Page

User can choose:
- Register
- Login

## 2. Registration

New user fills the registration form.
After successful validation:

- Registration number is generated
- Student data is saved
- User is moved to the welcome page

## 3. Welcome Page

User can see:
- Registration success message
- Generated registration number

User can continue to login.

## 4. Login

User enters:
- Registration number
- Password

After successful login, user is moved to the dashboard.

## 5. Dashboard

User can view:
- Profile photo
- Personal details
- Registration details

User can logout from the dashboard.

## 6. Forgot Password

User can reset password by verifying:
- Registration number
- Security question
- Security answer
