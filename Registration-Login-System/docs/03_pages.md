# Frontend Pages Information

This project contains six HTML pages.

## 1. index.html

Purpose:

This is the home page of the application.

It provides options for:
- New student registration
- Existing student login

## 2. registration.html

Purpose:

This page is used to create a new student account.

Information collected:
- First name
- Last name
- Date of birth
- Email
- Mobile number
- Password
- Security question
- Security answer
- Profile photo

## 3. welcome.html

Purpose:
This page shows registration success information.
It displays:
- Welcome message
- Generated registration number
The user can continue to login.

## 4. login.html

Purpose:
This page is used for student login.
User enters:
- Registration number
- Password
After successful login, user moves to the dashboard.

## 5. forget-password.html

Purpose:
This page helps users reset their password.
User verifies:
- Registration number
- Security question
- Security answer
After verification, a new password can be created.

## 6. dashboard.html

Purpose:
This page displays student profile information after login.
Information shown:
- Profile photo
- Student name
- Registration number
- Email
- Mobile number

It also provides logout option.
