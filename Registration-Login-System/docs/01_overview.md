# Project Overview

## Project Name

Student Registration and Login System

## About Project

This is a frontend learning project created to practice HTML, CSS, Bootstrap and JavaScript.
The project allows students to register, login and view their profile information.

## Purpose

The main purpose of this project is to understand:

- Form creation
- Form validation
- JavaScript events
- Local Storage
- JSON data handling
- Image upload handling

## Technologies Used

- HTML5
- CSS3
- Bootstrap
- JavaScript
- JSON
- Local Storage

## Main Features

- Student registration
- Automatic registration number generation
- Login using registration number and password
- Forgot password option
- Student dashboard
- Profile photo display
- Client-side validation

## Project Type

Frontend only application.
No backend server or database is used.
Data is stored in browser Local Storage.

## Future Improvement

In future, this project can be connected with a backend API and database for real user management.
