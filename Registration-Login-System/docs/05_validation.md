# Validation Rules

Validation is used to check user input before saving data.

## Registration Page

### Name
- Only alphabets are allowed
- Length should be between 2 and 30 characters

### Date of Birth

- Date cannot be a future date
- Minimum age: 16 years
- Maximum age: 110 years

### Email
- Email format should be valid
- Duplicate email is not allowed

### Mobile Number

- Only 10 digit numbers are allowed
- Duplicate mobile number is not allowed

### Password

Password should contain:
- Minimum 8 characters
- One uppercase letter
- One lowercase letter
- One number
- One special character

### Confirm Password
- Password and confirm password should match

### Security Details

- Security question is required
- Security answer is required

### Profile Photo

Allowed:
- JPG
- JPEG
- PNG

Size:

- Minimum: 20 KB
- Maximum: 1 MB

## Login Page
Validation:

- Registration number is required
- Password is required
- User credentials must match stored data

## Forgot Password Page

Validation:
- Registration number check
- Security question check
- Security answer check
- New password validation
